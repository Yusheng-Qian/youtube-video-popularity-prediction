# YouTube Video Popularity Prediction

This project builds machine learning models to predict YouTube video popularity using engagement metrics and metadata features.

## Overview

The goal of this project is to understand the key drivers behind video popularity and build predictive models estimating potential video views.

Machine learning models are trained using engagement metrics such as:

- likes
- comments
- publishing time
- video metadata

## Methods

Models implemented:

- Linear Regression
- Random Forest

Feature engineering and model interpretation were performed using SHAP to identify important predictors of video performance.

## Results

The Random Forest model demonstrated stronger predictive performance.

Key drivers of video popularity include:

- engagement metrics
- publishing timing
- video metadata features

## Interactive App

A Streamlit application was built to simulate how different video attributes impact predicted video views.

## Tools

- Python
- Pandas
- Scikit-learn
- SHAP
- Streamlit
